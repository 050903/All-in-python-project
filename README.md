## Django Blog Application Advanced

A full-featured blog platform built with Django, featuring:

- **Post Categories:** Group and filter posts by category.
- **Search:** Search posts by title or content.
- **Comments:** Users can comment on posts.
- **Pagination:** Posts are split into pages for easy navigation.
- **Modern UI:** Responsive Bootstrap design, navbar, styled forms, and flash messages.

**Directory:**  
`Django Blog Application Advanced/`

**How to run:**
```bash
cd "Django Blog Application Advanced"
python manage.py migrate
python manage.py runserver
```
Visit [http://127.0.0.1:8000/](http://127.0.0.1:8000/) in your browser.

## Requirements

- Python 3.8+
- See `requirements.txt` for all dependencies.
- For Django projects:  
  - Django 5.x
  - Bootstrap (via CDN in templates)
- For utility scripts:  
  - `requests`, `beautifulsoup4`, `pytube`, `yt_dlp`, etc.

Install dependencies with:
```bash
pip install -r requirements.txt
```

## How to Run

1. **Django Apps:**  
   - Navigate to the app directory.
   - Run migrations: `python manage.py migrate`
   - Start the server: `python manage.py runserver`

2. **Utility Scripts:**  
   - Run directly with Python:  
     `python 1.py`  
     `python 3.py`  
     etc.

3. **Web HTML App:**  
   - Open `quantum_calculator_web.html` in your browser.
